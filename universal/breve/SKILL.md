# /breve — Resposta em 3-5 linhas

## Regra única

Quando o Valter usar `/breve`:

1. **Máximo 3 a 5 linhas.** Conta linhas, não palavras.
2. **Não simplifica conteúdo.** Mantém precisão técnica/estratégica.
3. **Sem markdown pesado.** Sem tabelas, sem listas longas. Bullet curto se ajudar.
4. **Sem intro nem despedida.** Já vai direto.
5. Se o pedido genuinamente NÃO cabe em 5 linhas → entrega 5 linhas + "Tem mais detalhe se quiser, fala".

## Combina

- `/breve /ghost` → 3-5 linhas cruas
- `/breve /framework` → blocos curtos nomeados, mesmo limite de linhas
